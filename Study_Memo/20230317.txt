	 HTML (Hyper Text Mark-up Language) : 하이퍼 텍스트 마크업 언어
	 	Hyper Text : 링크(Link, 웹페이지에서 다른 페이지로 이동할 수 있도록 하는 것)
	 	Mark-up Language : 태그(Tag)를 사용해서 문법을 표기하는 언어
	 	프로그래밍 언어 (X) -> 표현 언어 (O) : 브라우저 화면에 뭔가 출력하는 기능만 있는 언어

			인터프리터 언어 : 한줄씩 실행 -> 에러가 나도 몰라요... -> 결과는 어떻게 될지 아무도 몰라...

	 		시작태그, 종료태그만 있는 경우도 있음

	 		*** 자바때처럼 가독성을 위해서 남발 X !
